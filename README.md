# PDF Splitter

A lightweight, browser-based PDF splitting tool that runs entirely in your browser without requiring server uploads.

## Features

- 📄 **Upload & Split**: Select a PDF and split it into multiple files
- 🔒 **Password Protected**: Support for encrypted PDFs
- 💾 **Multiple Formats**: Download individual files or all as a ZIP archive
- 🚀 **Fast & Secure**: Processing happens entirely in your browser
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile devices
- 🎨 **Modern UI**: Clean glass-morphism design with intuitive controls

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Upload a PDF file
4. Select your split options (by page range or automatic splits)
5. Download the resulting files

## How to Use

1. **Upload**: Tap the upload area or drag & drop a PDF file (max 100MB)
2. **Configure**: Choose your split method and configure options
3. **Process**: Click split to process your PDF
4. **Download**: Get your files individually or as a ZIP

## Technical Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with glass-morphism effects
- **JavaScript**: Client-side PDF processing
- **pdf-lib**: PDF manipulation library
- **jszip**: ZIP file creation
- **Font Awesome**: Icon library
- **Google Fonts**: Inter typeface

## Browser Support

Works on all modern browsers that support:

- ES6 JavaScript
- Fetch API
- FileReader API
- Web APIs for blob handling

## File Limits

- Maximum PDF size: 100MB
- Supports encrypted/password-protected PDFs
